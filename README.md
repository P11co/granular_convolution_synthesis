# GCS: Granular Convolution Synthesis

## Try GCS out
### 1. Installation
- [ ] conda create --name gcs python=3.12
- [ ] conda activate gcs
- [ ] pip install -r requirements.txt

### 2. Running the script
- [ ] python granular_convolution_synthesis.py
